Verzweigungen mit if / else

Bisher wurde jede Anweisung deiner Programme genau einmal ausgeführt, von oben nach unten. In echten Programmen müssen Entscheidungen getroffen werden: Nur wenn eine bestimmte Bedingung zutrifft, soll ein Codeblock laufen. Genau das leisten Verzweigungen.

Die if-Anweisung

Eine Bedingung ist ein Ausdruck, der entweder true oder false ergibt. Der Block in den geschweiften Klammern läuft nur, wenn die Bedingung true ist.

Volljaehrig.java 
int alter = 18;

if (alter >= 18) {
    System.out.println("Volljährig");
} else {
    System.out.println("Minderjährig");
}
Gut zu wissen

Der else-Zweig ist optional. Er fängt alle Fälle ab, in denen die Bedingung false ist.

Vergleichsoperatoren

Diese Operatoren liefern einen Wahrheitswert und bilden das Herz jeder Bedingung:

==
gleich
!=
ungleich
>= <=
größer/kleiner gleich

Gerade oder ungerade?

Das Programm soll für die Zahl in zahl ausgeben, ob sie gerade oder ungerade ist.

Erwartete Ausgabe
7 ist ungerade
Tipp

Mit dem Modulo-Operator % erhältst du den Rest einer Division. zahl % 2 == 0 ist genau dann wahr, wenn zahl gerade ist.

public class Main {
    public static void main(String[] args) {
        int zahl = 7;

        if (zahl % 2 == 0) {
            System.out.println(zahl + " ist ungerade");
        } else {
            System.out.println(zahl + " ist gerade");
        }
    }
}
